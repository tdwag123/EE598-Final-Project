# Sum of Two Squares in Lean 4

In this project we implement and partially verify a **constructive algorithm** for the sum-of-two-squares problem in Lean 4, covering:

- A custom **Gaussian integer** with Euclidean GCD.
- A **five-stage algorithm**: factorize `n`, check solvability, represent each prime power, combine with Brahmagupta–Fibonacci.
- Partial proofs (generated by Claude) for an **Ouput Guarantee** and a **Solvability Criterion** theorem relating the algorithm to Fermat's criterion.

---

## The Algorithm

**Fermat's theorem:** `n` is a sum of two squares iff every prime `p ≡ 3 (mod 4)` appears to an even power in `n`.  For example: `65 = 5 × 13`, both `≡ 1 (mod 4)`, so `65 = 4² + 7²`.

Each prime power `pᵉ` is represented separately, then combined:

```lean
-- p = 2: recurrence (a−b, a+b) gives a²+b² = 2ᵉ
-- p ≡ 3: trivial (pᵉ/², 0) since e must be even
-- p ≡ 1: find t²≡−1 (mod p), then gcd(p, t+i) in ℤ[i], then raise to e
def combine (x y : SqPair) : SqPair :=   -- Brahmagupta–Fibonacci is the same as taking N(x·y) = N(x)·N(y) for Gaussian Integers
  let (a,b) := x; let (c,d) := y
  (a*c - b*d,  a*d + b*c)

def findSumOfTwoSquares (n : Nat) : Option SqPair :=
  if ¬ isSolvable n then none
  else some ((primeFactorization n)          -- [(2,3),(3,2),(5,1)] for n=360
    .map (fun (p,e) => reprPrimePower p e)   -- represent each pᵉ as (a,b) with a²+b²=pᵉ
    |>.foldl combine (1,0))
```

Each `reprPrimePower p e` gives a pair `(a,b)` whose norm `a²+b²` equals `pᵉ`.
`combine` merges two such pairs into one whose norm is the **product** of the two norms, folding over all prime powers chains the norms together: `p₁^e₁ · p₂^e₂ · … = n`.
The final pair `(x,y)` is the decomposition `x²+y²=n`.

For `p ≡ 1 (mod 4)` there is no closed-form — instead, we find `t` with `t² ≡ −1 (mod p)`, so `p | (t+i)(t−i)` in `ℤ[i]` but `p` divides neither factor alone. Computing `gcd(p, t+i)` in `ℤ[i]` extracts the shared factor `a+bi`, and its norm `a²+b² = p` gives the representation:

```lean
structure GaussianInt where re : Int; im : Int
def norm (z : GaussianInt) : Int := z.re * z.re + z.im * z.im
partial def gcd (a b : GaussianInt) : GaussianInt :=  -- Euclidean algorithm for Gaussian integers
  if b == 0 then a else gcd b (a - gaussianDiv a b * b)
```

## Future Work: Proving the Following

**Output Guarantee**
```lean
theorem findSumOfTwoSquares_outputGuarantee (n : Nat) (x y : Int) :
    findSumOfTwoSquares n = some (x, y) → x * x + y * y = (n : Int)
```

**Solvability Criterion**
```lean
theorem findSumOfTwoSquares_solvability (n : Nat) :
    isSolvable n = true ↔ ∃ x y : Int, x * x + y * y = (n : Int)
```
